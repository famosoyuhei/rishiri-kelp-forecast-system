## チェックリスト

### A. 334地点マーカーの表示

- [ ] 331干場（H_XXXX_XXXX）と3特別地点（A_/R_）が地図上に全件表示されるか
- [ ] 特別地点3つが正しい座標に表示されるか

  - `A_2417_1867`（本泊アメダス）: 北緯45.2417°、東経141.1867°
  - `R_1800_2392`（利尻山頂）: 北緯45.1800°、東経141.2392°、標高1,721m
- [ ] 干場と特別地点でマーカーのアイコン・色が視覚的に区別できるか

- [ ] 地図タイル（OpenStreetMap）が正常に読み込まれるか

### B. 干場カード（クリック時のポップアップ）

- [ ] 干場名（H_XXXX_XXXX）と所属（町 > 地区 > 部落）が表示されるか
- [ ] 干場ニックネームが設定されている場合、カード内に正しく表示されるか
- [ ] ニックネームに「この呼び名はこの端末だけに保存されます」の説明文があるか
- [ ] 7日間の乾燥スコアが日付ごとに視覚的に表示されるか
- [ ] 乾燥記録ボタン（記録追加）が干場（H_）のみに表示され、特別地点（A_/R_）には出ないか
- [ ] 削除ボタンが特別地点に表示されないか（システム保護）
- [ ] onshore wind判定の表示がある場合、角度差（wind_mountain_angle_diff）が表示されるか

### C. 階層的フィルタリング（町 → 地区 → 部落）

- [ ] 「利尻町」「利尻富士町」の絞り込みが正常に機能するか

- [ ] 特別地点（R_1800_2392 は町/地区/部落=―）がフィルタ適用時に正しく扱われるか

### D. 利尻島伝統風名（rishiri_wind_names.js）

- [ ] 16方位の全風名が正しく対応づけられているか
- [ ] 風名の表示が気象データ取得失敗時でも壊れないか（フォールバック表示）

### E. UIの視認性・操作性

- [ ] 日本語フォントが文字化けせず表示されるか
- [ ] スマホ（375px幅想定）で地図とマーカーが操作しやすいか
- [ ] 高密度エリア（沓形周辺など）でマーカーが重なって選択できない問題はないか
- [ ] ダークモード環境でマーカーや文字が見えなくなる問題はないか

## 報告形式

問題を発見した場合は以下の形式で報告してください：

```
【重大度】🔴高（リリースブロッカー） / 🟡中（リリース前に修正推奨） / 🟢低（改善提案）
【カテゴリ】A〜Eのどれか
【該当箇所】ファイル名:行番号（例: kelp_drying_map.html:1234）
【問題内容】何が起きているか
【再現手順】どうすれば再現できるか
【修正提案】どう直せばよいか
```

問題がない場合は `✅ [カテゴリ名] 異常なし` と記載してください。
