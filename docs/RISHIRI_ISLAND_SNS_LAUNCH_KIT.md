# 利尻島内向けSNS初回配信セット

**目的**: 利尻島内の昆布関係者に、利尻島昆布干場予報システムを過度に広げすぎず届ける。  
**前提**: 事実確認は `README.md`、`kelp_drying_map.html`、`start.py`、`docs/MARKETING.md` に基づく。  
**注意**: 予報は作業判断の補助情報。必ず当たる、収益が上がる、公式・公認である等の表現は使わない。
**期間**: 島内向け宣伝は昆布漁期に合わせ、9月末で一旦終了する。

---

## 1. Facebook / Instagram広告文 3案

### 案1: いちばん狭く届ける

**見出し**  
利尻島で昆布を干す方へ

**本文**  
明日の干場ごとの乾燥予報を、スマホで確認できます。  
利尻島の331か所の干場と3つの観測/基準点をもとに、7日間の乾燥適性を表示します。  
まずは自分の地区・干場を開いて、明日の判断材料として見てください。

**CTA**  
干場予報を見る

**想定画像**  
アプリの地図画面 + 「利尻島で昆布を干す方へ」の文字

---

### 案2: 前日の段取り

**見出し**  
明日の昆布干し、夕方に確認

**本文**  
「明日は干せそうか」を、干場ごとに確認できる利尻島向けの予報アプリです。  
降水量・湿度・風速などをもとに、7日間の乾燥適性を表示します。  
経験と勘を置き換えるものではなく、前日の段取りを決めるための判断材料です。

**CTA**  
明日の予報を見る

**想定画像**  
スマホで7日間予報を開いている画面

---

### 案3: 家族・共有向け

**見出し**  
家族で同じ干場予報を確認できます

**本文**  
利尻島の昆布干し向けに、干場ごとの乾燥予報を確認できるページを公開しています。  
地図から干場を選ぶと、7日間の予報と乾燥スコアを見られます。  
家族や作業を手伝う方との共有にも使えます。

**CTA**  
使い方を見る

**想定画像**  
QRコード + スマホ画面 + 地区名

## 6. 配信時の注意

- 広告対象は利尻町・利尻富士町を最優先にする。
- 地域指定が難しい場合は、利尻島を覆う最小半径にする。
- 観光向け写真ではなく、干場・アプリ画面・地名を前面に出す。
- 「無料アプリ」だけを目立たせない。
- 最初のCTAは「登録」より「自分の干場を見る」にする。
- コメントで「当たるの？」と聞かれたら、断定せず「判断材料として使ってください」と返す。
