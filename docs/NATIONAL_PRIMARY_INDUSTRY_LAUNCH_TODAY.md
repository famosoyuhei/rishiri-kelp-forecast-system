# 全国一次産業向け 初回発信セット

作成日: 2026-06-27 JST

## 目的

利尻島昆布干場予報システムを「実例」として、全国の天候で作業判断が変わる一次産業向けに、相談・ヒアリング前提の専用アプリ導入支援を伝える。

島内向けの無料公開とは混ぜない。全国向けは地域・業種・地点・作業判断が現場ごとに違うため、相談なしの汎用提供として書かない。

## 今日の優先順位

1. Xで固定投稿候補を出す
2. Xで単発投稿を1本出す
3. Threadsで柔らかい背景投稿を1本出す
4. note記事の下書きを作る
5. 全国向け相談フォームを別途用意する

## X 固定投稿候補

```text
天気予報を見るだけでは、一次産業の作業判断には足りないことがあります。

刈るか。
干すか。
防除するか。
収穫するか。
船を出すか。

必要なのは天気そのものではなく、現場ごとの作業可否です。

北海道・利尻島では、331か所の昆布干場と3つの観測/基準点を扱い、7日間の乾燥予報を確認できるWebアプリを作りました。

この考え方は、牧草乾燥、露地野菜の防除、果樹の霜害・高温対策、養殖や沿岸作業の強風判断にも応用できます。

全国向けには、相談・ヒアリング前提で個別構築します。
地域、地点、作業手順、通知先、記録項目が現場ごとに違うためです。

まずは小さく、1か月試験運用できる形から設計します。
```

## note 構成

1. 天気予報は便利になったが、現場では「で、今日は作業するのか」が残る
2. 利尻島の昆布干しで感じた、場所ごとの差
3. 331か所の干場と3つの観測/基準点を扱うアプリにした理由
4. 経験と勘を置き換えるのではなく、判断材料を増やす
5. 全国の一次産業に応用できること
6. 導入支援では、最初に何をヒアリングするか
7. 小さく試験運用して、記録から改善する

## 相談導線の文言

全国向けは、利尻島内向けの無料利用フォームとは分ける。

```text
全国向けの導入支援は、業種・地域・地点・作業判断が現場ごとに違うため、相談とヒアリングを前提にしています。

「うちの作業にも使えるかもしれない」と思った方は、作業内容・地域・困っている判断を教えてください。
```

## 全国向け相談フォームで聞く項目

- お名前 / 法人名
- 業種
- 地域
- 作業内容
- 天候で迷う判断
- 見たい地点数
- 通知したい相手
- 現在使っている天気情報
- 記録したい項目
- 相談希望の有無
- 返信先メール

## 禁止表現

- 必ず収益改善します
- 完全自動で判断できます
- 全国どこでも同じアプリで使えます
- 昆布干しアプリをそのまま売ります
- 気象予報士レベルの判断を代替します

## 使ってよい表現

- 作業判断の補助
- 相談・ヒアリング前提
- 小さく試験運用
- 地点ごとに見る
- 記録して翌年の改善材料にする
- 現場の経験と勘を否定しない
