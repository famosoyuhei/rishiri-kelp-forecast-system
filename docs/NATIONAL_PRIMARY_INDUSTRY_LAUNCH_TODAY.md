# 全国一次産業向け 初回発信セット

作成日: 2026-06-27 JST

## 目的

利尻島昆布干場予報システムを「実例」として、全国の天候で作業判断が変わる一次産業向けに、相談・ヒアリング前提の専用アプリ導入支援を伝える。

島内向けの無料公開とは混ぜない。全国向けは地域・業種・地点・作業判断が現場ごとに違うため、相談なしの汎用提供として書かない。

## note 構成

1. 天気予報は便利になったが、現場では「で、今日は作業するのか」が残る
2. 利尻島の昆布干しで感じた、場所ごとの差
3. 331か所の干場と3つの観測/基準点を扱うアプリにした理由
4. 経験と勘を置き換えるのではなく、判断材料を増やす
5. 全国の一次産業に応用できること
6. 導入支援では、最初に何をヒアリングするか

フォームや投稿の近くに、次の文言を入れる。

```text
フォーム送信、内容確認、メールでの簡単な初回回答までは無料です。

個別ヒアリング、要件整理、試作、構築、運用支援などが必要になった場合は、希望する関わり方と内容を確認したうえで事前に相談します。フォーム送信だけで料金が発生することはありません。
```

## 連絡・打ち合わせ手段

全国向けは相談・ヒアリングが前提になるため、電話番号を載せずに打ち合わせできる手段を確認する。

フォームでは次を選べるようにする。

- WhatsApp
- Zoom
- Google Meet
- LINE
- Skype
- メール
- チャット・文字でのやり取りのみ
- 電話以外は使えない
- まだわからない

運用方針:

- できればWhatsAppを第一候補にする
- 相手がSkypeしか使えない場合も対応する
- 電話番号での通話対応は基本的に行わない
- 電話以外が使えない人には、メール等の文字でのやり取りを案内する
- WhatsApp、LINE、Skype等のIDはフォーム初回入力では必須にせず、初回返信後に必要に応じて確認する

## 有料支援スタイル

基本は、相談しながら作って運用後も見ていく伴走型を標準にする。

ただし、相手の体制や予算によって、次の入口を用意する。

| スタイル | 向いている相手 | 内容 |
|---|---|---|
| 伴走型 | 不安なく進めたい法人・団体 | ヒアリング、要件整理、試作、構築、修正、運用相談まで継続して支援 |
| 丸投げ型 | 現場要望は出せるが技術作業を任せたい相手 | 要望を聞いて、使える状態に近い形までこちらで構築 |
| コード納品型 | 自分や社内に設置できる人がいる相手 | 完成コードや設定手順を渡し、設置・運用は相手側で実施 |
| 自走支援型 | Claude CodeやCodex等で自分で作りたい相手 | 最初の使い方、設計の考え方、詰まりやすい点だけ教えて、その後は自走 |

投稿やフォームでは、価格を断定しない。
「希望する支援スタイルにより変わるため、内容確認後に相談」と書く。

- お名前 / 法人名

- 作業内容
- 天候で迷う判断

- 希望する連絡・打ち合わせ手段
- 現在使っている天気情報
- 記録したい項目
- 相談希望の有無
- 返信先メール

## 禁止表現

- 必ず収益改善します
- 完全自動で判断できます
- 全国どこでも同じアプリで使えます
- 昆布干しアプリをそのまま売ります
- 気象予報士レベルの判断を代替します

## 使ってよい表現

- 作業判断の補助
- 相談・ヒアリング前提
- 小さく試験運用
- 地点ごとに見る
- 記録して翌年の改善材料にする
- 現場の経験と勘を否定しない
