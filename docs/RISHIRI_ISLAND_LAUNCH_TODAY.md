# 利尻島内向け 今日使う公開セット

公開URL:
https://rishiri-kelp-forecast-system.onrender.com/island

目的:
利尻島内で昆布を干す方、家族、手伝いの方に、干場ごとの7日間乾燥予報を知ってもらう。

注意:
予報は作業判断の補助情報です。必ず当たる、公式、公認、収益が上がる、といった表現は使いません。
