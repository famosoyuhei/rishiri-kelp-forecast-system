## 1. アプリを開く

**スマホ・タブレットの場合**

1. Safariまたはブラウザ（Chrome など）を開く
2. アドレスバーに以下のURLを入力してください

```
https://rishiri-kelp-forecast-system.onrender.com
```

3. 地図の画面が表示されれば完了です

> **注意**: 初回の表示に少し時間がかかる場合があります（最大30秒ほど）。  
> Renderの無料プランを使用しているため、しばらく誰も使っていないと「スリープ状態」になることがあります。  
> 画面が真っ白なときは、少し待ってから再読み込みしてください。

## 2. スマホのホーム画面に登録する（オススメ）

ホーム画面に登録すると、アプリのように一発で開けます。  
**インターネットなしでも使える**ようになります。

### iPhoneの場合（Safari使用）

1. Safariでアプリを開く
2. 画面下の **「共有」ボタン**（四角に上矢印のマーク）を押す
3. 「**ホーム画面に追加**」を押す
4. 「追加」を押して完了

### Androidの場合（Chrome使用）

1. Chromeでアプリを開く
2. 画面右上の **「︙」（縦三点ボタン）** を押す
3. 「**ホーム画面に追加**」を押す
4. 「追加」を押して完了

---

## 3. 画面の見方

### メイン画面（地図）

アプリを開くと地図が表示されます。

| 表示されているもの | 意味 |
|-----------------|------|
| **色のついた丸いマーク** | 干場の場所。色が乾燥のしやすさを表しています |
| **緑色のマーク** | 条件良好（干せそう） |
| **黄色のマーク** | 条件普通 |
| **赤色のマーク** | 条件不良（干しにくい） |
| **灰色のマーク** | データなし |

### 地図の操作

| 操作 | やり方 |
|------|--------|
| 拡大する | 画面を2本指で広げる、またはマーク右上の「＋」ボタン |
| 縮小する | 画面を2本指で狭める、またはマーク右上の「－」ボタン |
| 地図を移動する | 1本指でドラッグ（なぞる） |
| 干場の詳細を見る | 丸いマークをタップ（押す） |

### 画面上部のボタン

| ボタン | 機能 |
|--------|------|
| **今日・明日・...（日付）** | 見たい日の予報に切り替える |
| **絞り込み** | 町・地区・部落で表示する干場を絞り込む |

## 4. 乾燥予報を確認する

### 特定の干場の予報を見る

1. 地図上の見たい干場（丸いマーク）を**タップ**する
2. 画面下に干場の情報が表示される
3. 「**7日間予報**」を見る

### 予報の見方

各日の予報には以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **スコア（点数）** | 100点満点。高いほど乾燥しやすい |
| **降水量** | 0mmであることが必須条件 |
| **最低湿度** | 94%以下であれば乾燥可能 |
| **平均風速** | 2.0m/s以上であれば乾燥可能 |
| **判定** | 「条件良好」「条件普通」「条件不良」など |

> **3つの絶対条件**  
> 降水量0mm ＋ 最低湿度94%以下 ＋ 風速2.0m/s以上  
> この3つが揃った日は乾燥可能と判断されます。

### 複数の干場をまとめて比較する

1. 「**絞り込み**」ボタンで自分の地区・部落を選ぶ
2. その地区の干場が地図上に表示される
3. マークの色で一目で比較できる

## 5. 干場を追加する

自分の干場がまだ登録されていない場合に追加できます。

1. 地図上の追加したい場所を**長押し**する
2. 「**この場所に干場を追加**」というボタンが表示される
3. ボタンを押す
4. 地形の種類（海岸・森林・平地など）を選ぶ
5. 「**追加する**」を押して完了

> 追加した干場には自動で名前（H_XXXX_XXXXという形式）がつきます。  
> **ニックネーム**を設定することで、自分だけの呼び名をつけられます（→[干場ニックネームの設定](#干場ニックネームの設定)）。

### 干場ニックネームの設定

1. 干場のマークをタップして詳細を開く
2. 「**ニックネーム**」の欄を探す
3. 自分がわかりやすい名前を入力する（例：「家の前」「浜田さんの横」）
4. 「**保存**」を押す

> ニックネームはこのスマホ・タブレットだけに保存されます。  
> 他の人の画面には表示されません。

---

## 6. 干場を削除する

> **注意**: 以下の条件の干場は削除できません。
> - 乾燥記録が入力されている
> - お気に入りに登録されている
> - 通知設定で使用されている
> - 他の人が同時に編集中（5分間）

### 削除の手順

1. 削除したい干場のマークをタップする
2. 詳細画面で「**削除**」ボタンを探す
3. 「**本当に削除しますか？**」という確認が出る
4. 「**はい**」を押して削除完了

---

## 7. 乾燥記録を入力する

実際に昆布を干した結果を記録できます。記録が増えるほど予報の精度が上がります。

### 記録の入力手順

1. 記録したい干場のマークをタップする
2. 「**記録を入力**」ボタンを押す
3. 以下を入力する：

| 入力項目 | 内容 |
|---------|------|
| **日付** | 昆布を干した日 |
| **結果** | 「完全乾燥」「部分乾燥」「中止」から選ぶ |
| **沖止め** | 沖止めの日は「沖止め」を選ぶ |

4. 「**保存**」を押して完了

> 過去の記録も入力できます。記録はサーバーに保存されるため、他の端末でも同じ記録が見られます。

## 8. お知らせ（通知）を設定する

翌日の乾燥予報を自動でお知らせします。

### 通知の種類

| 通知 | タイミング | 内容 |
|------|----------|------|
| **夕方通知** | 毎日16:00 | 翌日の乾燥予報 |
| **早朝通知** | 毎日01:30 | 当日の乾燥予報（最新版） |
| **緊急アラート** | 随時 | 予報スコアが30点以上急変したとき |

### 通知の設定手順

1. 画面右上の「**🔔 通知**」ボタンを押す
2. 初回は「免責事項への同意」が表示される → 内容を確認して「**同意する**」を押す
3. ブラウザから「通知を許可しますか？」と聞かれたら「**許可**」を押す
4. 通知を受け取りたい干場の「**通知ON**」ボタンを押す

### 沖止め日の設定

沖止め日（出漁しない日）には通知が来ないよう設定できます。

1. 通知設定画面を開く
2. 「**沖止め日設定**」を探す
3. 沖止めの日付を入力する

### 通知が届かない場合

- スマホの設定でこのアプリ（ブラウザ）の通知が「オフ」になっていないか確認する
- iPhoneの場合: 設定 → Safari → 通知
- Androidの場合: 設定 → アプリ → Chrome → 通知

## 9. チャットボットに質問する

画面右下の「**💬**」ボタンを押すと、干場アシスタントに質問できます。  
インターネット接続は必要ですが、AIは使用していないため**追加費用はかかりません**。

### 質問の例

| 質問の種類 | 例 |
|----------|-----|
| 今日の予報 | 「今日干せますか？」「今日の天気は？」 |
| 明日の予報 | 「明日は干せそうですか？」 |
| 地区の予報 | 「沓形で干せる日は？」「仙法志は今週どうですか？」 |
| 複数エリアの比較 | 「沓形と鴛泊を比べて」「東側と西側どちらがいい？」 |
| 気象の説明 | 「湿度って何？」「風速2m/sってどのくらい？」 |
| アプリの使い方 | 「干場の追加方法は？」「記録の入力方法は？」 |

> 「初心者向けに説明して」「詳しく教えて」と付け加えると、  
> やさしく丁寧な説明に切り替わります。

## 10. インターネットが使えないときも使う（オフライン）

ホーム画面に登録済みの場合（→[2章参照](#2-スマホのホーム画面に登録するオススメ)）、  
インターネットなしでも一部機能が使えます。

### オフラインでできること

- 地図の表示（最後にオンラインで開いたときのキャッシュ）
- 過去に確認した干場の情報の閲覧
- 記録の入力（次にオンラインになったとき自動で保存されます）

### オフラインでできないこと

- 最新の気象データの取得
- 新しい予報の確認

> 画面上部に「**オフライン**」と表示されているときは、  
> 表示されているデータが最新でない可能性があります。

## 11. 困ったときは

### よくある質問

**Q: 地図が真っ白で何も表示されない**  
A: インターネット接続を確認してください。接続できていれば、少し待ってから再読み込み（画面を引っ張る、またはブラウザの更新ボタン）してみてください。

**Q: 干場のマークが表示されない**  
A: 地図の拡大・縮小を試してください。また、「絞り込み」で特定の地区が選ばれている場合は解除してみてください。

**Q: 通知が来ない**  
A: スマホの通知設定を確認してください（→[通知が届かない場合](#通知が届かない場合)）。

**Q: 干場を削除しようとしたら「削除できません」と出る**  
A: 記録が入力されている干場は削除できません（→[6章参照](#6-干場を削除する)）。

**Q: 予報と実際の天気が違う**  
A: 予報はOpen-Meteo（気象モデル）のデータを使用しています。予報は必ずしも実際の天気と一致しません。記録を蓄積することで予報精度が向上します。

**Q: 昆布シーズンの見通しを知りたい**  
A: 画面上部の「**シーズン予報**」タブを押してください。月別の見通し・ENSO状況が確認できます。

## 付録: 地点名（H_XXXX_XXXX）について

このアプリでは干場に自動で名前がつきます。

| 種類 | 形式 | 意味 |
|-----|------|------|
| 干場 | `H_XXXX_XXXX` | 緯度・経度から自動生成 |
| アメダス沓形 | `A_1783_1383` | 気象庁の公式観測点 |
| アメダス本泊 | `A_2417_1867` | 気象庁の公式観測点 |
| 利尻山頂 | `R_1800_2392` | 風向の基準点 |

わかりにくい場合はニックネームを設定してください（→[干場ニックネームの設定](#干場ニックネームの設定)）。
