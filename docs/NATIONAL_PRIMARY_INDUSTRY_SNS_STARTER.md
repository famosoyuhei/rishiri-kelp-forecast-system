# 全国一次産業向けSNS初回発信セット

**目的**: 利尻島昆布干場予報システムの実例をもとに、全国の天候で作業判断が変わる一次産業へ「専用アプリ導入支援」を伝える。  
**主な媒体**: X、Threads、note。  
**注意**: 利尻島内向け告知とは混ぜない。導入効果や収益改善を断定しない。全国向けは十人十色の現場ニーズに合わせた個別構築が前提のため、相談・ヒアリングを必須導線にする。

---

## 1. X 単発投稿案

### 案1: 問題提起

```text
天気予報を見るだけでは、一次産業の作業判断には足りないことがあります。

「刈るか」
「干すか」
「防除するか」
「出漁するか」

必要なのは天気そのものではなく、現場ごとの作業可否。

利尻島の昆布干場予報アプリをもとに、相談・ヒアリング前提の天候判断アプリ導入支援を始めます。
```

### 案2: 利尻の実例

```text
利尻島向けに、昆布干場ごとの7日間乾燥予報アプリを作っています。

331か所の干場と3つの観測/基準点を扱い、降水・湿度・風速などから乾燥適性を表示します。

この考え方は、牧草乾燥、露地野菜の防除、果樹の霜害、養殖の強風判断にも応用できます。
```

### 案3: 導入支援

```text
一次産業向けの天候判断アプリ導入支援でできること。

まず現場ごとの作業判断を相談で伺います。

・作業判断基準のヒアリング
・地点マスター作成
・気象API連携
・作業可否スコア
・LINE/メール通知
・作業記録
・Google Sheetsで精度分析

「天気を見る」から「判断を残す」へ。
```

## 5. note 冒頭下書き

```markdown
天気予報は毎日見られるようになりました。

雨雲レーダーも、週間予報も、風の予報も、スマホで簡単に確認できます。

それでも一次産業の現場では、最後に残る問いがあります。

「で、今日は作業するのか」

牧草を刈るのか。昆布を干すのか。露地野菜の防除に入るのか。果樹の霜対策をするのか。船を出すのか。

現場で必要なのは、天気そのものだけではなく、その天気を作業判断に翻訳することです。

私はその一つの実例として、北海道利尻島の昆布干場予報システムを作っています。

利尻島では、同じ島の中でも干場の場所によって風や湿度、雨の影響が変わります。一般的な天気予報だけでは、「自分の干場で明日干せるか」までは判断しにくい場面があります。

そこで、利尻島内の331か所の昆布干場と、3つの観測/基準点を扱い、干場ごとに7日間の乾燥予報を確認できるアプリを作りました。

このアプリは、経験と勘を置き換えるものではありません。

むしろ、現場の判断を支える材料を増やし、記録を残し、次の年の改善に使えるようにするためのものです。
```
