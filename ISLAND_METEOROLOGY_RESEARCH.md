## 1. 利尻島の地理的特性と気象的意義

| 項目 | 値 |
|-----|-----|
| 位置 | 北緯45°N、東経141°E（日本海北部） |
| 島の直径 | 約20 km |
| 面積 | 約182 km² |
| 山頂（利尻山） | 標高1,721 m |
| 海岸線形状 | ほぼ円形（最短半径9 km〜最長半径11 km） |
| 周辺海域 | 日本海（西〜南）、宗谷海峡（北東） |
| AMeDAS観測点 | 沓形（SW, 14 m）・鴛泊（NE, 39 m） |

利尻島は「海上に突出した孤立円錐峰」という形態の典型例です。  
島の面積に対して山頂標高が極めて高く（アスペクト比 h/D ≒ 0.086）、同様の形態を持つ世界の島は**済州島・マデイラ島・テネリフェ島・ハワイ島**などに限られます。  
この形態は以下の3つの気象効果を同時・複合的に発生させます。

1. **フロード数支配の流れ分岐** — 風速・大気安定度次第で「島を越える流れ」と「島を回る流れ」が切り替わる
2. **全周海岸での海陸風収束** — 島を取り囲む海面から昼間に同時に海風が発生し、島中心部で収束する
3. **後流渦（カルマン渦列）の形成** — 逆転層が山頂より低い条件下で大気境界層が島に遮られ、渦列を形成する

## 2. フロード数と流れの体制

### 2-1. 定義と計算式

```
Fr = U / (N × h)
```

| 記号 | 意味 | 代表値 |
|-----|------|--------|
| U | 島に吹きつける気流の水平風速 [m/s] | 3〜15 m/s |
| N | ブラント・ヴァイサラ振動数 [s⁻¹] | 0.008〜0.012 s⁻¹ |
| h | 障害物（島）の高さ [m] | 1,721 m |

**ブラント・ヴァイサラ振動数 N の目安**

| 大気状態 | N [s⁻¹] |
|---------|---------|
| 乾燥した安定大気（冬季） | 0.010〜0.012 |
| 湿潤・弱安定（夏季海洋性） | 0.006〜0.008 |
| 強い逆転層内部 | 0.020〜0.030 |

### 2-2. 利尻山（h = 1,721 m）でのFr計算例

| シナリオ | U [m/s] | N [s⁻¹] | Fr | 流れの体制 |
|---------|---------|---------|-----|----------|
| 夏季・微風 | 3 | 0.009 | **0.19** | 強いブロッキング |
| 夏季・中程度 | 8 | 0.009 | **0.52** | 部分ブロッキング |
| 冬季モンスーン | 15 | 0.010 | **0.87** | 遷移域（越山主体） |
| 冬季強風 | 20 | 0.010 | **1.16** | 越山支配 |

### 2-3. 流れの体制と予報特性

| Fr値 | 体制 | 干場への影響 |
|------|------|------------|
| > 2.0 | **越山支配** | 山頂越えの重力波・乱流、風下海岸に強い下降流・昇温・乾燥 |
| 1.0〜2.0 | **遷移域** | 越山と回り込みが混在、山頂に笠雲 |
| 0.4〜1.0 | **部分ブロッキング** | 分流高度より下の気流が島を回る、海岸角での風加速 |
| 0.2〜0.4 | **強ブロッキング・渦形成域** | 後流渦発生の可能性、島風上に停滞点、海岸コーナーで5〜10 m/s加速 |
| < 0.2 | **完全ブロッキング** | ほぼ全気流が迂回、島の真後ろに静穏域 |

### 2-4. 分流高度（Sheppard公式）

気流が「島を越える部分」と「島を回る部分」を分ける境界面の高度：

```
Hs / h = 1 − Fr
```

例：Fr = 0.3 の場合、**Hs = 1,721 × 0.7 ≒ 1,205 m**  
→ 地上から1,205 mより下の気流は利尻山を「回り込む」

この高度以下の気流が海岸の乾燥条件に直接影響するため、frの実時間モニタリングは予報精度の要になります。

## 3. 地形性降水と風上・風下非対称

### 3-1. 機構

卓越気流が山岳斜面に当たると強制上昇 → 断熱冷却 → 凝結 → 降水（風上側で雨雪）。  
山頂を越えた気流は乾燥断熱減率（9.8°C/km）で降下・昇温し、風下側は乾燥・晴天となる。

### 3-2. 世界事例の降水比（風上：風下）

| 島 | 山頂高度 | 卓越風 | 風上:風下 降水比 | 備考 |
|----|---------|--------|--------------|------|
| 利尻島 | 1,721 m | 冬:NW、夏:SW | 推定 3〜5倍 | AMeDAS2点が直接計測 |
| 済州島 | 1,950 m | 冬:NW | 2〜3倍 | 台風時に地形性増幅を確認 |
| レユニオン島 | 3,070 m | 東南東貿易風 | **約6倍** | 世界降水記録保有（24h: 1,825 mm） |
| テネリフェ島 | 3,715 m | 北東貿易風 | 3〜4倍 | 北東350〜700 mm vs 南西<300 mm/年 |
| ハワイ島 | 4,207 m | 北東貿易風 | **5〜10倍（最大50倍）** | 14気候区のうち10区が同一島に存在 |
| マデイラ島 | 1,862 m | 北東貿易風 | 2〜3倍 | 渦列のモデルケース |

### 3-3. 利尻島の季節別降水パターン

**冬季（11〜3月）**  
- 卓越風：北西（シベリア高気圧〜東アジア冬季モンスーン）  
- 風上（北西〜北）：鴛泊北部・本泊・富士岬が多雪  
- 風下（南東側）：沓形・仙法志方面が降水影 → 乾燥した晴天が出やすい

**夏季（6〜9月）**  
- 卓越風：南〜南西（太平洋高気圧の西縁循環）  
- 風上（南西側）：沓形・仙法志方面に霧雨・にわか雨  
- 風下（北東側）：鴛泊・ペシ岬周辺がやや乾燥  
- **昆布干しシーズンと一致**: 干場選択に風上・風下の把握が直結

### 3-4. AMeDAS2点観測の活用

沓形（南西, 14 m）と鴛泊（北東, 39 m）の2点は、利尻島の支配的な降水非対称軸をちょうど対角で押さえています。  
2点間の降水差（沓形 − 鴛泊）の符号で、その時刻の「降水風上方向」が推定できます。

```
(沓形降水 − 鴛泊降水) > 0  →  南西側が風上 → 南西〜南系の風
(沓形降水 − 鴛泊降水) < 0  →  北東側が風上 → 北西〜北系の風
```

## 4. フェーン現象

### 4-1. 機構

**乾燥フェーン（メカニカルフェーン）**  
- 気流が乾燥断熱減率（9.8°C/km）で山を越え降下する純粋な昇温。  
- 利尻山（1,721 m）を越える場合: 最大 **約17°C** の気温上昇が理論値。

**湿潤フェーン（クラシックフェーン）**  
- 風上側では湿潤断熱減率（約6°C/km）で上昇 → 降水で水分を失う。  
- 山頂越え後は乾燥断熱減率（9.8°C/km）で降下 → 正味で **3〜4°C/km** 分のフェーン昇温。

### 4-2. 利尻島でのフェーン発生パターン

| 季節 | 吹送方向 | フェーン側 | 典型昇温量 | 昆布干しへの影響 |
|-----|---------|----------|----------|--------------|
| 冬〜春 | 北西 | 沓形（南東側） | 5〜15°C | 急激な高温乾燥 → 干し過ぎ注意 |
| 夏 | 南西 | 鴛泊（北東側） | 3〜8°C | 乾燥促進、ただし継続時間短い |

**北西日本でのフェーン被害事例**（EurekAlert, 2021）：  
冬季に太平洋側から山脈越えで日本海側に達する気流が、記録的な高温・乾燥風害を引き起こした事例が文献化されています。  
利尻島のスケールは小さいものの、山頂での分水嶺効果は同様の物理過程が作動します。

## 5. 海陸風・谷風・山風（日変化風系）

### 5-1. 円形島特有の海陸風

孤立した円形島では、海風が**全方位から同時に島中心へ収束**します。

1. **午前中**: 島の山腹が加熱 → 斜面に沿った上昇気流（谷風・山谷風）が発達
2. **日中〜午後**: 海面より島陸上が高温 → 全周から海風が吹き込み、島中心部で収束上昇
3. **夕方〜夜**: 陸上の急冷却 → 山頂から海岸へ向けた下降風（山風・カタバ風）が全周に発達
4. **深夜〜夜明け**: 陸風が島から海へ向けて吹き出す

### 5-2. 海風収束降水

島の面積が約200 km²以上になると、対向する海風前線の衝突が午後の対流性降水を誘発します。  
利尻島（約182 km²）はこの閾値の境界域にあります。  
参考（Sobel et al. 2011）：直径20 kmの熱帯島でも、対流解像モデルで午後の降水ピークが再現されています。

島の直径 D ≒ 20 km、海風速 v ≒ 3〜5 m/s として、  
**海風前線の衝突まで約0.5〜1.5時間**（前線が島中心に到達する時間）。  
→ 海風の開始時刻（概ね10〜11時）から加算すると、**収束による降雨リスクは11〜13時**ごろ。

### 5-3. カタバ風（夜間斜面下降風）

清天夜に地表面が長波放射で冷却されると、冷気が斜面に沿って海岸へ向けて流れ下ります。

| パラメータ | 代表値 | 計算根拠 |
|-----------|-------|--------|
| カタバ風速 | 2〜8 m/s | 斜面傾斜・安定度依存 |
| カタバ層厚 | h_layer ≒ 0.05 × Δz | 高度差1,000 mの斜面で約50 m層厚 |
| 最大発達時刻 | 深夜〜夜明け前（02〜05時 JST） | 放射冷却の累積後 |

**昆布干しへの示唆**:  
夜間〜早朝のカタバ風は「山から吹き下ろす乾燥した空気」であり、海岸部の湿度を一時的に低下させます。  
早朝01:30の通知スコアにおいて、弱いシノプティック場（<5 m/s）でも夜間湿度が低い場合はカタバ風効果と解釈できます。

### 5-4. 日変化サイクルのまとめ（夏季晴天日）

```
時刻 (JST)    地上風系                湿度トレンド    干し適性
────────────────────────────────────────────────
00〜05時      カタバ風（山→海岸）      低下傾向       良好
05〜08時      凪・無風                微低下          良〜普通
08〜11時      谷風開始（斜面→山頂）    横ばい          普通
11〜14時      海風収束（全周→中心）    上昇            注意（収束降水リスク）
14〜18時      海風最強（2〜5 m/s）    低下（乾燥風）  良好
18〜22時      海風→山風に移行         上昇→低下       普通〜良好
22〜24時      山風・カタバ風開始       低下            良好
```

## 6. 山岳島後流渦（カルマン渦列）

### 6-1. 形成条件

大気境界層の上端に**逆転層**があり、その逆転層の高さが島の山頂より低い場合、地表から逆転層底までの「スラブ」が島に遮られ、周囲を迂回します。迂回した気流が島の背後で合流し、**交互に渦を放出**する構造がカルマン渦列です。

形成に必要な条件（Madeira・済州・Guadalupe島での観測に基づく）：

1. **Fr < 0.4**（強いブロッキング）
2. 境界層の上端（逆転層底）が山頂高度以下
3. 上流風速がほぼ定常（大きなベア・バッキングなし）
4. ストルーハル数 **St = f × D / U ≒ 0.15〜0.22**（f は渦放出頻度 [Hz]）

### 6-2. 観測事例の定量データ

**マデイラ島（1,862 m, 33°N）** — Vortex Streets to the lee of Madeira, WCD 2023  
- 10年間の1 km解像度COSMOシミュレーション
- 渦放出のStrouhal数：St = 0.142〜0.166
- 渦の移流速度：上流風速の **約82%**
- 季節変化：4〜8月に最多（貿易風逆転層が山頂以下）、9月以降に逆転層が上昇して消失
- 反時計回り渦（低気圧性）の方がコリオリ力により長く持続

**済州島（1,950 m, 33°N）** — Wu et al. 2025, Earth and Space Science  
- 冬季北西モンスーン時に最多（MODIS画像での確認）
- 最適風速：18〜54 km/h（5〜15 m/s）
- 渦コア直径：20〜40 km
- 尾流長：200〜500 km

**カナリア諸島（テネリフェ等）** — NASA MODIS  
- 2015年5月20日に3島同時渦列をMODISが捉えた
- 渦列幅：100 km超、尾流長：数百 km

### 6-3. 利尻島での発生推定条件

| 条件 | 充足時期 |
|-----|---------|
| Fr < 0.4（U < 6.2 m/s @ N=0.009） | 夏季・春秋の穏やかな日 |
| 逆転層底が山頂（1,721 m）以下 | 夏季オホーツク高気圧下（850〜1,000 hPa逆転層） |
| 定常な南〜南西気流 | 7〜8月 |

→ **利尻島北方海域で夏季南西の穏やかな気流時に、後流渦列が発生している可能性が高い。**  
渦の存在は北側（鴛泊方面）の干場において「風向が予測と大きく異なる」「収束線通過に伴う突然の弱風→強風」という形で現れうる。

### 6-4. 予報への影響

- 後流渦発生時は**渦の位置によって風向が数時間で180°反転**することがある
- 渦内の下降流域では晴天・乾燥、上昇流域では雲・霧が発生
- 現行5 km格子のOpen-Meteo MSMでは渦を解像できないため、誤差源となる

## 7. 霧の形成メカニズム

### 7-1. 利尻島に関連する霧の種類

#### (a) 移流霧（最頻・最重要）

暖湿な気塊が冷たい海面上を移流し、露点温度以下に冷却されて発生。

- **発生時期**: 5〜8月（海面水温 < 気塊の露点温度）
- **気象場**: 親潮（オホーツク海流）の低SST + 南〜南東気流 + オホーツク高気圧
- **霧出現率**: 6〜8月、オホーツク海上で月平均50〜55%（JMA/海洋気象台資料）
- **指標**: 沓形AMeDASの **露点温度降下量（気温 − 露点温度）< 2°C** → 高霧リスク

#### (b) 地形性霧（斜面霧）

山腹に当たった湿潤気流が強制上昇し、断熱冷却で凝結。実質的に「低雲が山腹に接地した状態」。

- 利尻山の笠雲・霧帽がこれに相当
- 笠雲の出現は「翌日の天気悪化」の前兆として漁師の間でも認識
- 笠雲形成高度の気温・露点差をOpen-Meteoの圧力面データから推定可能

#### (c) 放射霧

夜間の長波放射冷却で地表面が冷え、接地気層が露点以下に下がって発生。

- 利尻島では地形がほぼ全面山腹のため、内陸盆地型の放射霧は発生しにくい
- 海岸平坦部（干場が集中）で発生することがある（清天・無風・高湿度の夜）

### 7-2. ENSOと霧頻度の関係

- **エルニーニョ年**: 日本海・オホーツク海の海面水温がやや高め → 気塊の冷却が弱まり移流霧減少 → 昆布干し条件が改善傾向
- **ラニーニャ年**: 海面水温が低め → 移流霧増加、6月〜7月の有効干し日数が減少傾向
- **中立年**: 平年並みの霧頻度

オホーツク高気圧の発達強度もENSOと連動し、強い年ほど南〜南東気流が卓越して移流霧頻度が増します。

## 8. 冬季—日本海海洋性降雪

### 8-1. 日本海海洋性降雪（Sea-Effect Snow）の機構

冬季シベリア高気圧からの冷た乾燥した北西季節風が相対的に暖かい日本海面上を吹き渡る際、海面からの顕熱・潜熱フラックスによって境界層が発達し、降雪雲帯（SES: Sea-Effect Snowstorms）が形成されます。

**日本海寒気移流収束帯（JPCZ）**  
シベリアから北朝鮮・中朝国境の山脈によって東西に分割された寒気が、日本海で再収束してつくる収束帯。この収束帯が利尻島に接触すると、**地形性上昇が加わった猛烈な降雪**が発生します。  
理論的最大降雪強度: **100 cm / 7時間**（水平収束 13 kg m⁻² h⁻¹に相当, BAMS 2020）

### 8-2. 利尻島における地形性降雪増幅

- 北西気流（Fr ≒ 0.8〜1.2）: 島を越える流れが支配的 → 島全体で地形性上昇降水
- 北西気流（Fr ≒ 0.3〜0.5）: 部分ブロッキング → 北側斜面（鴛泊方面）で多雪、南側（沓形方面）は比較的少雪
- JPCZの接触時: 急激な視程低下・激しい降雪 → 干場の昆布回収に支障

### 8-3. 降雪シーズンの干場管理への影響

冬季（昆布干しシーズン外）でも、海岸付近の干場設備（干し台・ロープ等）への雪荷重・凍結被害の予測に有用です。  
現行システムの予報閾値（降水量 0mm 絶対条件）は降雪も正しく排除できます。

## 9. 予報上の課題と対策技術

### 9-1. 根本的な解像度問題

| スケール | 代表距離 | 対応する気象現象 | Open-Meteo MSM（5 km）での捕捉 |
|---------|---------|----------------|-------------------------------|
| シノプティックスケール | >200 km | 高気圧・低気圧 | ◎ 十分 |
| メソβスケール | 20〜200 km | 利尻島全体の地形効果 | △ 部分的 |
| 局地スケール | 2〜20 km | 海岸角の収束・風加速 | × 困難 |
| マイクロスケール | <2 km | 個別干場の微気象 | × 不可能 |

### 9-2. フロード数リアルタイム診断

Open-Meteo APIの`pressure_level`パラメータから鉛直気温プロファイルを取得し、Nを計算することで、Fr診断を自動化できます。

```python
# 現行システム（start.py）への追加可能な疑似コード
def calculate_froude_number(sounding_data, h=1721.0):
    """
    sounding_data: 各圧力面の気温データ (hPa: temp_C のdict)
    """
    # 850-700 hPa間の鉛直温度勾配からNを推定
    T_850 = sounding_data.get(850, 5.0)  # °C
    T_700 = sounding_data.get(700, -5.0)  # °C
    z_850 = 1500.0  # m (概算)
    z_700 = 3000.0  # m (概算)
    dT_dz = (T_700 - T_850) / (z_700 - z_850)  # K/m (負値が安定)
    g = 9.81
    T_mean_K = (T_850 + T_700) / 2 + 273.15
    N2 = (g / T_mean_K) * (-dT_dz + 9.8e-3)  # 乾燥大気近似
    N = max(N2, 0) ** 0.5
    # 地上風速（850 hPa風を使用）
    U = sounding_data.get('wind_850', 8.0)
    if N < 1e-4:
        return 999.0  # 中立大気では定義なし
    return U / (N * h)
```

**Fr判定に基づく地形補正切替（目標実装）**:

| Fr | 適用補正モード |
|----|-------------|
| < 0.4 | 「回り込み」モード: 海岸角の風加速係数を1.3〜1.5に強化、後流域では0.5に低減 |
| 0.4〜1.0 | 標準補正（森林減衰・海岸増強、onshore判定付き） |
| > 1.0 | 「越山」モード: 風下海岸にフェーン乾燥補正、score+15点、降水を0補正 |

> **現行コード**: Frは未計算。補正は加算型固定値（森林−2.5 m/s、海岸+1.0 m/s）で常時適用。
> `850/700 hPaデータはすでに取得済み`のため、Fr計算のコード追加コストは小さい。

### 9-3. AMeDAS2点差分による即時バイアス補正

沓形と鴛泊の**観測値の比率**から、Open-Meteoの誤差を後補正できます。

```python
bias_wind = amedas_kutsugata_wind / openmeteo_kutsugata_wind
corrected_spot_wind = openmeteo_spot_wind * bias_wind
```

この手法は統計的ダウンスケーリングの一形態であり、地点別バイアス補正として文献化されています（cf. Reinecke & Durran 2008）。

### 9-4. 高解像度WRF導入の場合の設定指針

より本格的な数値ダウンスケーリングを行う場合の参考：

| パラメータ | 推奨値 | 根拠 |
|-----------|-------|------|
| 親領域 | 9 km格子 | JMA MSM相当 |
| 中間領域 | 3 km格子 | 島周辺の海岸線解像 |
| 利尻島領域 | 1 km格子 | 渦列・地形性降水を解像（Madeira研究より） |
| 地形データ | SRTM/GLO-90（90 m DEM） | 本システム採用済みソース |
| 積雲パラメータ | 3 km以下では明示対流（KF法不要） | 対流解像モデル要件 |

### 10-2. 利尻島と最類似の島：済州島

**共通点**:
- 緯度帯（33〜45°N）、孤立峰の直径・高度が近い
- 冬季北西モンスーン支配（対馬暖流/日本海 vs 黄海）
- 夏季移流霧（オホーツク高気圧 vs 黄海冷水域）
- カルマン渦列の観測事例が豊富

**相違点**:
- 済州島は島面積が約4倍（1,849 km²）で海陸風の強度が大きい
- 利尻島は孤立度が高く（周囲の島がない）、後流渦が乱されにくい
- 利尻島は高緯度なためコリオリパラメータが大きく、渦の非対称性が強い

### 10-3. マデイラ島事例の定量的教訓

マデイラ島（山頂1,862 m、直径57 km）での10年WRFシミュレーション（WCD 2023）から、類似する利尻島への転用が期待できる知見：

1. **逆転層高度が山頂以下に下がった日数**: 年間**約120〜150日**で渦列が形成
2. **渦列のStrouhal数 St = 0.15〜0.17** → 利尻（D ≒ 20 km, U ≒ 8 m/s）なら渦放出周期 ≒ **D/(St×U) ≒ 1,500秒（約25分）**
3. 1 km解像度以上でないと渦列は再現されない
4. 渦が消滅するのは逆転層が上昇して山頂を超えた時（9月以降）→ **利尻では9月以降の秋昆布干しシーズンは渦リスクが低下**
