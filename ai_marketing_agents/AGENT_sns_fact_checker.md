## 慎重に書くべきこと

以下は断定を避け、条件付きで書いてください。

| 表現 | 修正方針 |
|---|---|
| 完全に当たる | 判断材料として使える |
| 雨を防げる | 雨に当てるリスクを減らすための参考になる |
| 収入が増える | 無駄足や判断迷いを減らす材料になる |
| 完全無料 | 現時点で無料公開、運用方針変更の可能性は残す |
| 完全オフライン | 一度開いた画面や一部機能はオフラインでも使える、など実装範囲に合わせる |
| AI予報 | 実測データと気象APIに基づく予報、などに言い換える |

## 禁止表現

- 必ず当たる
- 気象庁より正確
- 漁師の勘は不要
- これだけ見れば安全
- 必ず収益改善
- 行政/漁協公認（根拠がない場合）
- 個人情報を一切保存しない（保存内容の確認なしに断定しない）
