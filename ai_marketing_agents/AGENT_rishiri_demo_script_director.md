## 作る成果物

### 1. 5分デモ台本

目的:

- まず使ってもらう
- QRを読み込んでもらう
- 自分の干場を見る

必ず含める流れ:

1. 共感から入る
2. アプリを開く
3. 自分の干場を探す
4. 明日と今週を見る
5. 通知をONにする
6. 「一週間だけ見てください」で終える

### 2. 15分説明台本

目的:

- 機能と信頼性を理解してもらう
- 家族や部落内で共有してもらう

含める内容:

- 干場ごとの予報
- 7日間予報
- 夕方通知・早朝通知
- 急変アラート
- 乾燥記録
- 島の風名
- 予報は補助情報であること

### 3. 30分説明会台本

目的:

- 漁協・部会・役場向け
- 導入・周知の判断材料を渡す

含める内容:

- なぜ一般天気予報だけでは足りないか
- 331干場+3観測/基準点の意味
- 実測データに基づく閾値
- 記録が増えるほど改善できる仕組み
- 免責と運用ルール
- 質疑応答

### 4. よくある質問と答え

- 当たらない日はどうする？
- 誰が管理している？
- 個人情報はある？
- 干場を追加できる？
- 通知を止められる？
- どのスマホで使える？

### 5. デモ前チェックリスト

- URL
- QRコード
- スマホ表示
- 通知説明
- 電波がない場合の代替
- 紙チラシ
- その日の予報確認
